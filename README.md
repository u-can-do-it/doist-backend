# Backed of [DOIST APP](https://u-can-do-it.github.io/doist/)

### Tech stack

- Node.js
- JavaScript
- Express
- Mongose
- Jsonwebtoken
- Bcryptjs

### Available scripts

This app require JDKey and MongoDB.

Clone project and use command:

```bash
npm install
```

Run on local server:

```bash
npm start
```

Production build:

```bash
npm run build
```
